# План внедрения глассморфизма в проект Prime Messenger

Этот план описывает шаги по обновлению дизайна приложения для достижения эффекта "стеклянного" интерфейса (glassmorphism), что придаст приложению современный и премиальный вид.

## Основные изменения

Глассморфизм характеризуется полупрозрачными фонами, размытием (blur), тонкими светлыми границами и использованием ярких цветов на фоне для подчеркивания эффекта стекла.

### 1. Ресурсы (Цвета и Drawables)
*   [NEW] `colors.xml`: Добавление новых цветов с альфа-каналом (полупрозрачный белый, светлые обводки).
*   [NEW] `bg_glass_card.xml`: Универсальный фон для карточек и панелей с закругленными углами и тонкой белой границей.
*   [NEW] `bg_main_gradient.xml`: Насыщенный градиентный фон, на котором эффект стекла будет выглядеть наиболее выигрышно.

### 2. Код (Расширения для размытия)
*   [MODIFY] `ViewExtensions.kt`: Добавление функции `applyGlassBlur()`. Для устройств на Android 12+ (API 31) будет использоваться системный `RenderEffect` для реального размытия заднего плана. Для старых версий будет использоваться полупрозрачность с обводкой.

### 3. Обновление экранов
*   **Экран входа и регистрации (`activity_login.xml`, `activity_register.xml`)**:
    *   Замена белого фона на насыщенный градиент.
    *   Помещение полей ввода в "стеклянные" панели.
    *   Обновление кнопок (стеклянный эффект или акцентный цвет с сиянием).
*   **Главный экран (`activity_chat_list.xml`)**:
    *   Обновление тулбара: реальное размытие контента при прокрутке.
    *   Улучшение кнопки "Начать общение" — придание ей вида матового стекла.
*   **Экран настроек (`activity_settings.xml`)**:
    *   Оформление элементов списка в виде стеклянных блоков.

### 4. Темы и стили
*   [MODIFY] `themes.xml`: Обновление основной темы `Theme.Prime` для поддержки новых визуальных эффектов.
